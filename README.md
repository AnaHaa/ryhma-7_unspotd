# Ryhma 7: UnSpotd

School project for course Mobiiliohjelmointi 2021

## Requirements

npm
expo-cli
Expo Go for mobile
React native tools
Prettier
VSCode

## Installation

Use the package manager [npm](https://www.npmjs.com/) to install UnSpotd.

```bash
npm i -g expo-cli
cd UnSpotd
npm install
expo start
```

Open the application on mobile using Expo Go and QR code or web application.
